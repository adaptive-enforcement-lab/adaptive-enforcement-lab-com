# Adaptive Enforcement Lab

Documentation site for the Adaptive Enforcement Lab organization.

**Live Site:** [adaptive-enforcement-lab.com](https://adaptive-enforcement-lab.com)

## Mission

To turn secure development into an enforced standard, not an afterthought.

## What's Here

Operator manuals and patterns for:

- GitHub App authentication and cross-repository automation
- File distribution workflows using matrix strategies
- Security-first CI/CD pipeline patterns

## Local Development

```bash
# Create virtual environment
python3 -m venv .venv
source .venv/bin/activate

# Install dependencies
pip install mkdocs-material

# Start dev server
mkdocs serve
```

Site available at `http://127.0.0.1:8000`
